# Koreader

## Overview

Koreader is a personalized Korean graded reading practice web app for an adult learner. It generates AI-powered reading passages tailored to the learner's level and interests, with word glosses, audio read-aloud, vocabulary lists, and a personal library.

## Stack

- **Monorepo tool**: pnpm workspaces
- **Node.js version**: 24
- **Package manager**: pnpm
- **TypeScript version**: 5.9
- **Frontend**: React + Vite (artifacts/koreader), served at `/`
- **API framework**: Express 5 (artifacts/api-server), served at `/api`
- **Database**: PostgreSQL + Drizzle ORM
- **Validation**: Zod (`zod/v4`), `drizzle-zod`
- **API codegen**: Orval (from OpenAPI spec)
- **AI**: OpenAI via Replit AI Integrations (gpt-5.2 for passage generation and word glossing)
- **State management**: Zustand (for unsaved generated passages)
- **Animations**: Framer Motion

## Features

1. **Home Dashboard** — stats, recent passages, quick-start topic chips
2. **Passage Generator** — topic, difficulty, length, reading style, optional vocab/grammar focus
3. **AI Generation** — real Korean passages generated by GPT-5.2, with sentences, vocabulary lists, images
4. **Passage Reader** — with 3 modes:
   - Reading Mode: clean Korean text only
   - Study Mode: translations hidden by default, expandable per sentence
   - Full View: everything visible
5. **Word Click Gloss** — click any Korean word for instant popup with meaning, romanization, grammar note, example
6. **Audio Read-Aloud** — browser TTS with Korean voice (ko-KR)
7. **Personal Library** — save, search, filter passages by difficulty / bookmark status
8. **Bookmarks** — favorite passages
9. **Vocabulary List** — per-passage vocabulary with romanization, part of speech, examples

## Key API Routes

- `POST /api/passages/generate` — Generate a new passage with AI
- `GET /api/passages` — List saved passages
- `POST /api/passages` — Save a passage to library
- `GET /api/passages/:id` — Get a single passage
- `DELETE /api/passages/:id` — Delete a passage
- `PUT /api/passages/:id/bookmark` — Toggle bookmark
- `POST /api/words/gloss` — Get word gloss (meaning, grammar note, example)

## Environment Variables

- `DATABASE_URL` — PostgreSQL connection string (auto-provisioned by Replit)
- `AI_INTEGRATIONS_OPENAI_BASE_URL` — OpenAI integration base URL (auto-set)
- `AI_INTEGRATIONS_OPENAI_API_KEY` — OpenAI integration API key (auto-set)
- `PORT` — Server port (auto-set by Replit)

## Running

Both workflows start automatically:
- `artifacts/koreader: web` — Vite dev server for the frontend
- `artifacts/api-server: API Server` — Express API server
