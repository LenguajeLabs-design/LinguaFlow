# SEO Strategy

## In scope
- Public marketing landing page (`/`)
- Public authentication entry points only where they affect crawlability or shared metadata (`/sign-in`, `/sign-up`)
- Shared HTML shell and public SEO infrastructure (`index.html`, robots.txt, sitemap.xml, llms.txt, favicon/social assets)

## Out of scope
- Authenticated and user-specific application routes (`/generate`, `/library`, `/library/:id`, `/passage`, `/favorites`, `/vocabulary`, `/settings`)
- Internal API routes under `/api/**`

## Target audience
- Adult learners using AI-assisted language reading practice, especially Korean learners.

## Primary keywords
- Korean reading practice
- Korean graded reader
- Korean reading app
- AI language learning app
